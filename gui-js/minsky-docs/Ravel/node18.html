Speed slider, time direction and model recording

These commands are specific to Minsky and are covered in detail in the Minsky component of this manual.

The speed slider controls the rate at which a model is simulated. The ``Reverse'' checkbox causes the simulation to run backwards rather than forwards in time.

The Record and Replay buttons respectively record the process of creating a model, and replay that process.
