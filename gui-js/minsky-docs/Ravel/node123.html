-th moment

The operator can be placed on the canvas in two ways:

  • From the Statistics (``statistics'') toolbar ; or
  • By typing the letters ``moment'' on the canvas and then pressing the Enter key
Returns the -th moment about the mean along a named dimension of all elements present. is specified by the numerical argument of the operation, which defaults to 1 (hence the result will be 0 in that case). If the dimension is not named, then the -th moment is over all elements present in the tensor. Note that missing elements are not counted.
