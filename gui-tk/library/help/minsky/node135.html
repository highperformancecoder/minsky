Ravel

Ravel is a skunkworks project enabling the interactive manipulation of multidimensional datacubes. This is a commercial technology, and you will need a license to use the software, as well as a copy of the Ravel plugin. This is still under active development, but you can read some more about it at Ravelation.

The Lock widget is used to fix the output of a Ravel at a particular point in time, making it easy to compare two different ravel settings.
